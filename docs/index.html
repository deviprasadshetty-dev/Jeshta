How it Works

Unlike simple Vector DBs, Delta uses a 4-stage pipeline to ensure Contextual Integrity.

1. Ingestion

Raw text is converted into an Immutable Atom. We strip noise and verifying semantic density.

2. Quantization

Embeddings are binarized. This allows us to use bitwise Hamming distance for searches, which is 10x faster than cosine similarity.

3. Delta Resolution

The magic sauce. We traverse the dependency graph to resolve conflicts (e.g., "I like red" -> "I actually like blue").

FAQ

Cloud or Local?

100% Local. Your data lives in `delta_mem.db` on your machine. No API keys required.

Does it work with LangChain?

Yes, but you don't need it. Delta is a standalone MCP server. It works with Claude Desktop, Cursor, and any MCP client.

How fast is it?

Retrieval takes < 15ms on standard hardware thanks to numpy-optimized bitwise operations.
